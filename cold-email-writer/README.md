# Cold Email Writer

AI-powered B2B cold email generator. Input your prospect info, get 3 personalized email versions instantly.

## Download

👉 **[Download ColdEmailWriter.exe](https://github.com/wf2023impr/B2B-Marketing-AI-Tools/releases)** — Double-click to run. No installation needed.

## Features

- **3 email styles** per generation: Direct, Story-based, Data-driven
- **Multi-provider**: OpenAI, Claude, DeepSeek — bring your own API key
- **Multi-language**: English, Chinese, Japanese, Korean, Spanish, French, German
- **Runs locally**: Your API key never leaves your machine

## How to Use

1. Double-click `ColdEmailWriter.exe` (or run `python app.py`)
2. A browser tab opens automatically
3. Select your AI provider and paste your API key
4. Fill in the prospect and product info
5. Click **Generate Emails**
6. Copy the version you like best

## For Developers

```bash
cd cold-email-writer
pip install -r requirements.txt
python app.py
```

### Build .exe yourself

```bash
python build.py
```
